## Sobre
Esse é o meu primeiro projeto completo em ReactJS feito totalmento do zero. Bem simples e intuitivo de se usar, ele tem como principal objetivo criar quizzes com o tema e as perguntas que o usuario desejar, e tambem de responder quizzes feitos por outras pessoas.

## Principais funcionalidades da aplicação:
- [x] Criar Quizzes
- [x] Responder Quizzes
- [x] Cadastro/Login(JWT)
- [x] Historico de Pontuações

## Para rodar o projeto 
- Clone o projeto
- Abra o terminal na pasta do projeto
- Instale as dependencias com `yarn` ou `npm install`
- Depois rode a aplicação com `yarn dev` ou `npm run dev`
